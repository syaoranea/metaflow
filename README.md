# MetaFlow Ng 🚀

MetaFlow is a modern application designed to help users organize their financial goals, track expenses, manage active debts, and review strategic plans. Built with Angular and styled using Tailwind CSS, MetaFlow offers a responsive, fast, and feature-rich user interface.

## ✨ Features

- **Dashboard & Strategic Review**: Keep track of your financial goal progress and access AI-powered suggestions.
- **Cash Flow & Debt Management**: Track active debts, record payments, and monitor your cash flow reports.
- **Scenario Simulator**: Simulate different scenarios by adapting resource inputs.
- **Reporting**: Detailed charts and options to download reports.
- **Modern UI**: A responsive, intuitive, and accessible experience seamlessly styled with Tailwind CSS.

## 🛠️ Tech Stack

- **Framework**: [Angular 18](https://angular.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Language**: TypeScript

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Version 18 or above recommended)
- `npm` or `yarn`

### Installation

1. Install all required dependencies:
   ```bash
   npm install
   ```

### Development Server

To start the development server, run:
```bash
ng serve
```
Navigate your browser to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

### Build for Production

To build the project for production, run:
```bash
ng build
```
The optimized build artifacts will be stored in the `dist/` directory, ready to be deployed.

## 🧪 Testing

### Unit Tests
Run `ng test` to execute the unit tests via [Karma](https://karma-runner.github.io).

### End-to-End Tests
Run `ng e2e` to execute the end-to-end tests (requires setting up a testing platform first).
